# What is Dead By Daylight?

Dead by Daylight is an asymmetric 4v1 multiplayer game where one horror Killer
tries to defeat four cunning Survivors. This Manual apworld focuses on the
Killer side of the experience.

# What does randomization do to this game?

Your access to Perks and Killers is restricted, until you or other players find
them as items in the multiworld. Your locations are granted when winning a game
while using a given Killer or Perk.

## About Winning

Dead by Daylight is an asymmetric, 4v1, multiplayer game where victory can mean
many things. Since this is a manual, you are free to choose your own win
condition. However, the recommended win condition works like this:

* You win if the survivors did not complete all generators. This implies that
  either nobody escaped or one person escaped via the hatch.
* Otherwise, you need a certain number of hooks to win.  This starts at 12 and
  is reduced by 1 each time you receive a "-1 Hook Required For Victory" item.
  This means if you receive 4 of them, you can even win by 2-hooking everyone
  and letting them escape!

# What is the goal of this randomizer?

The goal is to win with a certain number of killers. The number of killers
required for victory can be selected in the settings file.

# Which items can be in another player's world?

Killers are shuffled, and can end up in any player's world.

If Perk Shuffle is enabled, Perks will also be sent to other worlds.

# How is DLC handled?

For the sake of convenience, each Killer can be turned off in the settings
file. This will also remove their Perks from the game if Perk Shuffle is
enabled.

If you don't own a DLC, simply turn off its Killer in the settings file.
